## Overview

This repository contains the source code for Jesse Yang's personal portfolio website. Built with modern web technologies, this site showcases Jesse's journey as a Full-Stack Developer, featuring his projects, work experience, education, and technical skills.

## Key Features

- **Modern Design**: Clean, intuitive, and visually appealing interface for a seamless browsing experience.
- **Fully Responsive**: Optimized for all devices and screen sizes, ensuring consistent usability everywhere.
- **Performance Optimized**: Streamlined code and assets for fast load times and smooth navigation.
- **Comprehensive Portfolio**: Showcases projects, work experience, education, and technical skills.
- **Accessibility Focused**: Inclusive design practices to ensure the site is usable by everyone.
- **Advanced Tech Stack**: Developed with TypeScript, React, Gatsby, Three.js, and more.

## Getting Started

To set up and run the project locally:

1. **Clone the repository:**
   ```sh
   git clone git@github.com:JesseYeung/Jesse-Yang-Portfolio.git
   ```
2. **Navigate to the project directory:**
   ```sh
   cd Jesse-Yang-Portfolio
   ```
3. **Install dependencies:**
   ```sh
   yarn install
   ```
4. **Start the development server:**
   ```sh
   yarn dev
   ```
5. **Open your browser and visit:** [http://localhost:8000](http://localhost:8000)

## Environment Variables

The project supports optional environment variables for search engine indexing features. Create a `.env.development` file (for development) or `.env.production` file (for production) in the root directory:

```env
# Site Configuration (required)
GATSBY_SITE_URL=your-site-url.com
GATSBY_ORGANIZATION=Your Organization Name

# Google Indexing API (optional - enables automatic Google search indexing)
GOOGLE_CLIENT_EMAIL=your-service-account@project.iam.gserviceaccount.com
GOOGLE_PRIVATE_KEY="-----BEGIN PRIVATE KEY-----\nYOUR_PRIVATE_KEY_HERE\n-----END PRIVATE KEY-----\n"

# IndexNow API (optional - enables automatic indexing on Bing, Naver, Seznam, Yandex)
INDEXNOW_KEY=your-indexnow-key-here

# Google Tag Manager (optional)
TAG_ID=GTM-XXXXXXX
```

**Note:** If these environment variables are not set, the corresponding indexing features will be skipped during the build process without causing errors.
